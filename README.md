# CwbChat
*an im code by caowenbo*
---
* 功能 ： 连接服务器，实时接收和发送消息，可登录，拉取在线好友，聊天记录持久化储存，与服务器通过心跳保持在线状态。
* 实现：通过tcp连接与服务器连接，消息在结构体中发送，本地使用sqlite保存聊天记录
* 跨平台 目前测试在win10以及linux下都可正常运行
* 编译环境 qtcreator5.12 qmake版本:4.9
